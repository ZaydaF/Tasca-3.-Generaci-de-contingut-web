# Conceptes bàsics

## Què és un DER?

Un diagrama Entitat Relació és una representació gràfica utilitzada per al disseny conceptual de bases de dades.

## Elements principals

- Entitats
- Atributs
- Relacions
- Cardinalitats

## Avantatges

- Facilita el disseny de bases de dades
- Permet detectar errors abans de la implementació
- Millora la comprensió del sistema

!!! info "Què és un DER?"
    Un diagrama Entitat-Relació permet representar visualment les dades d’un sistema abans de crear la base de dades.


!!! warning "Important"
    No s’ha de començar a crear taules sense haver analitzat abans les entitats, atributs i relacions.

???+ question "Pregunta de repàs"
    Quina diferència hi ha entre una entitat i un atribut?

??? success "Resposta"
    Una entitat és un objecte o concepte del sistema; un atribut és una característica d’aquesta entitat.

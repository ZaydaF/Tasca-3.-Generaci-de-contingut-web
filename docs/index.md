# Bases de Dades

## Diagrames Entitat Relació (DER)

Esta web conté una introducció als diagrames Entitat Relació utilitzats en el disseny de bases de dades.

Els DER permeten representar visualment les entitats, atributs i relacions d’un sistema d’informació.

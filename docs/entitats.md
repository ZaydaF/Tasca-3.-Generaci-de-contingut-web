# Entitats i atributs

## Entitats

Les entitats representen objectes o conceptes del món real.

Exemples:
- Alumne
- Professor
- Curs

## Atributs

Els atributs descriuen característiques de les entitats.

Exemples:
- Nom
- DNI
- Telèfon

## Claus primàries

Cada entitat ha de tindre un atribut identificador únic.
